MOHGCN-MOHE Multi-Omics Fusion Model

Model Architecture Overview

The MOHGCN-MOHE multi-omics fusion model integrates data from multiple genomic modalities through a novel approach that combines heterogeneous graph neural networks and hypergraph neural networks. The architecture consists of these key components:

  • Multi-Omics Data Input: Processes raw data matrices from mRNA expression, DNA methylation, and miRNA expression.
  • Data Processing: Constructs two complementary graph structures:
    • Hyperedge Groups: Forms hypergraphs for each omics type with a special fused hyperedge group connecting information across modalities.
    • Heterogeneous Graphs: Builds heterogeneous graphs capturing relationships between samples and features/genes.
  • Multi-Omics Fusion Encoding: Unique encoders (MOHE-MOHGCN) that simultaneously leverage three input types:
    • Omics-specific hypergraphs capturing high-order relationships
    • Heterogeneous graphs capturing sample-feature interactions
    • Fused hypergraph enabling cross-omics information sharing
  • Feature Distribution: Encoded features represent omics-specific patterns enhanced by both graph structures.
  • Multi-Omics Self-Attention: Applies multi-head attention mechanism to integrate information across omics types.
  • Tensor Correlation Pooling (TCP): Learns confidence weights for each omics modality based on task relevance.
  • Final Label Prediction: Produces classification outputs for clinical applications.
